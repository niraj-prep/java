## Getting Started

To run any of the programs in this repository locally on your machine:

### Prerequisites

- **Java Development Kit (JDK):** Make sure you have JDK 8 or a newer version installed.
- **IDE / Editor:** You can use IntelliJ IDEA, Eclipse, VS Code, or a simple text editor with terminal access.

### Compilation and Execution

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/niraj-prep/java.git
   cd java/src
   ```
2. Compile a specific Java file using the `javac` command:
   ```bash
   javac ProgramName.java
   ```
   _(E.g., `javac Calculator.java`)_
3. Run the compiled class file using the `java` command:
   ```bash
   java ProgramName
   ```
   _(E.g., `java Calculator`)_

## Concepts Covered

Throughout these assignments, the following concepts are practiced:

- **Encapsulation & Abstraction** (using standard modifiers, getters, and setters)
- **Inheritance & Polymorphism** (multilevel inheritance, method overriding/overloading)
- **Constructors & `this` / `super` keywords**
- **String Handling** (String, StringBuilder, StringBuffer)
- **Exception Handling** (try, catch, finally, custom exceptions)
- **Multithreading** (Thread priority, basic thread management)
- **File & I/O processing** (Basic file extensions and content processing)
